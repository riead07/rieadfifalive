# RieadLiveFIFA ⚽🔥

A premium, secure, and private live web dashboard designed to host your personal live casting feeds from OBS Studio. It features private room access via license keys, device-limit enforcement, and a comprehensive admin panel.

---

## 🔒 Security & Authentication Architecture

1. **User Room Access (`/login`)**:
   - Access to the live stream page `/` is restricted. Users must enter a valid, active **License Key** generated by the administrator.
   - Restricts concurrent usage per key based on **Max Devices**. If a key is configured for 2 devices, a 3rd device will be blocked from logging in until one logs out.
   
2. **Admin Command Center (`/admin` & `/admin/dashboard`)**:
   - Secure login via passkey: **`riead07@#%`**
   - Manage license keys: Create, Block/Unblock, and Delete keys dynamically.
   - Tracks live active session counts (e.g. `Devices: 2/5`) for each terminal key.

## 🛠️ Step-by-Step Deployment Guide

### Part 1: Push updates to GitHub
Your code has already been successfully committed and pushed to your GitHub repository:
`https://github.com/riead07/rieadfifalive`

### Part 2: Deploying to Render
1. Go to [Render Dashboard](https://dashboard.render.com/) and log in.
2. Click **New +** > **Web Service**.
3. Select and connect your repository: `rieadfifalive`.
4. Use these settings:
   - **Name**: `rieadfifalive`
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `node server.js`
5. Click **Deploy Web Service**.

### Part 3: Connecting Your Custom Domain (`www.rieadfifa26live.com`)
1. In the Render Web Service page, go to **Settings** on the left menu.
2. Scroll down to the **Custom Domains** section.
3. Click **Add Custom Domain** and enter `rieadfifa26live.com` (and `www.rieadfifa26live.com`).
4. Log into your domain registrar (GoDaddy, Namecheap, Cloudflare, etc.) and add the following DNS records:
   - **Type**: `CNAME` | **Host/Name**: `www` | **Value/Target**: `rieadfifalive.onrender.com`
   - **Type**: `ANAME`/`ALIAS` or redirect | **Host/Name**: `@` | **Value/Target**: `rieadfifalive.onrender.com`
5. Click **Save** in Render. Render will issue a free SSL Certificate and bind your website!

## 🎮 How to Stream from OBS to your Web Room
1. Open **OBS Studio**.
2. Go to **Settings** > **Stream**.
3. Set **Service** to **Twitch** (recommended for zero proxy/geo-blocking issues). Connect your account or paste your stream key, and click **Start Streaming**.
4. Visit your website `www.rieadfifa26live.com`.
5. Enter your created license key to enter the room.
6. Open the tab **Twitch Live Embed**, type your Twitch username, and click **Connect Twitch Stream**. Your OBS stream will load instantly!
